# Motion Graphic Designer Portfolio

Modern portfolio website with smooth animations and 3D effects.

## 🚀 Tech Stack

* Next.js
* React
* TypeScript
* Tailwind CSS
* Three.js
* GSAP
* Framer Motion

## ✨ Features

* Premium UI Design
* 3D Hero Animation
* Smooth Scroll Effects
* Fully Responsive
* Contact Form
* Fast Performance

## 📦 Installation

```bash
npm install
npm run dev
```

## 🛠 Build

```bash
npm run build
npm start
```

## 📁 Main Files

* `src/app/page.tsx` – Home Page
* `src/data/portfolio.ts` – Content Data
* `src/components/` – UI Components
* `src/app/api/` – Contact API
